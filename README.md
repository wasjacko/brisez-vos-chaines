# Brisez vos chaînes

Page lock-screen iPhone : 13 chaînes 3D en Three.js qui enchaînent une sphère. Chaque tâche cochée brise la chaîne correspondante. Les 13 brisées → sphère dorée, « LIBÉRÉ ».

## Stack

- Next.js 15 + React 19
- Three.js (scène 3D raw, sans react-three-fiber)
- Tailwind CSS 4
- TypeScript strict

## Dev

```bash
pnpm install
pnpm dev
```

Ouvre http://localhost:3000.

## Deploy

Déployé sur Vercel.
